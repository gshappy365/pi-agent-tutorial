# 附录

## 关于附录

本附录提供了三部分实用内容，供学习过程中随时查阅。

| 章节 | 内容 | 适用场景 |
|------|------|----------|
| **术语表** | 按拼音首字母排序的核心术语解释 | 遇到不熟悉的概念时查阅 |
| **常见问题** | 开发过程中的典型错误及解决方案 | 部署和开发遇到问题时查阅 |
| **参考资源** | 官方文档链接和推荐阅读 | 想深入了解某个主题时查阅 |
| **部署指南** | 将教程站点部署到 Vercel 的完整步骤 | 准备发布教程站点时查阅 |

## 使用建议

- **学习过程中**：遇到术语不理解，先看术语表
- **开发过程中**：遇到报错信息，先查常见问题
- **深入学习时**：参考资源中的官方文档是最权威的学习材料
- **发布站点时**：按照部署指南一步步操作
