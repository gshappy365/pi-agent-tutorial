## 小结

这些资源涵盖了教程涉及的各个技术领域。建议优先阅读官方文档，它们是最权威、最及时的学习材料。遇到问题时，可以按以下顺序查找：

1. 本教程的 FAQ
2. 对应技术的官方文档
3. GitHub Issues / Stack Overflow

## 小练习

1. 浏览 OpenAI 的 Function Calling 文档，了解本教程之外的更多高级用法
2. 阅读 Anthropic 的 Tool Use 文档，对比其与 OpenAI 的实现差异
3. 查看 VitePress 的部署指南，了解除了 Vercel 之外的其他部署方式
